neoagent
====================================

==================
Introduction
==================

neoagent is a memcached protocol proxy server daemon.

==================
Features
==================

- connection pooling
- configuration with JSON
- fail over with backup server function
- support some memcached command(get, set, add, delete, incr, decr, quit)
