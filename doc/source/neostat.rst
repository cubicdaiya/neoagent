neostat
====================

==================
DESCRIPTION
==================

neostat is a status monitor for neoagent

==================
OPTIONS
==================

**\-h, --host**

 hostname

**\-p, --port**

 stat port number

**\-s, --sock**

 stat socket path

**\-o, --once**

 if this option specified, neostat gets stats only once

**\--help**

 show help
